# 🌴 PALM ÉLYSÉE – Luxury Hotel Website

Projeto de website institucional desenvolvido para um hotel de luxo fictício chamado **PALM ÉLYSÉE**.  
O objetivo é transmitir sofisticação, elegância e exclusividade por meio de um design moderno e refinado.

---

## ✨ Sobre o Projeto

O PALM ÉLYSÉE é um site desenvolvido com foco em:

- Identidade visual premium
- Layout elegante e minimalista
- Experiência do usuário (UX)
- Responsividade
- Interações suaves com JavaScript

Este projeto simula o site oficial de um hotel de alto padrão, destacando acomodações, serviços e atmosfera exclusiva.

---

## 🚀 Tecnologias Utilizadas

- **HTML5** – Estrutura semântica do projeto
- **CSS3** – Estilização, layout e animações
- **JavaScript** – Interatividade e efeitos dinâmicos

---

## 🎨 Funcionalidades

- Layout responsivo
- Efeitos de hover em imagens
- Animações suaves
- Navegação estruturada
- Design sofisticado com paleta premium (preto carvão e dourado)
